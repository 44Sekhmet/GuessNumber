# 猜數字遊戲 🎯

一個簡單又好玩的猜數字小遊戲！試著在最少次數內找出 1 到 100 之間的神秘數字。

🔗 **[立即遊玩](https://guessnumber-c057.onrender.com/)**

---

## 🕹️ 遊戲玩法

1. 遊戲會隨機產生一個 **1 到 100** 之間的神秘數字
2. 在輸入框中輸入你的猜測，按下 **確定**
3. 系統會根據你的猜測即時縮小範圍，顯示目前的**下限**與**上限**
4. 持續猜測，直到你找出正確答案為止
5. 答對後畫面會顯示你總共猜了幾次 🏆

---

## 💡 小技巧

- 建議使用**二分法**：每次都猜目前範圍的中間值，可以最快速地縮小範圍
- 例如：範圍是 1～100，先猜 **50**；若太小，再猜 **75**；若太大，再猜 **62**……以此類推
